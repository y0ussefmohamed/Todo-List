# Todo List App

A simple and efficient Todo List iOS application built with SwiftUI and Core Data.

## Description

This application allows users to manage their daily tasks effectively. It uses Core Data for persistent storage, ensuring that your tasks are saved even when the app is closed. The user interface is built using SwiftUI, providing a modern and responsive experience.

## Features

- **Add Tasks**: Easily add new tasks to your list.
- **Edit Tasks**: Update existing tasks as needed.
- **Delete Tasks**: Remove tasks that are completed or no longer needed.
- **Persistent Storage**: All data is stored locally using Core Data.
- **Clean UI**: A minimalist design focused on usability.

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.0+
